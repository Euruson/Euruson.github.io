Pelican and voce

在使用Pelican搭建自己的博客网站时，遇到了非常多的问题。网上关于Pelican的中文教程很少，有些问题和配置在说明文档里也是一带而过（可能是我太菜了吧）。这里列举一些问题并提供一些解决方案供。 关于Fabric脚本 因为我是在Win10环境下配置的，要用make的话有点麻烦，于是选择使用Fabric。 我用的是Python 3.6.1，旧版的Fabric不兼容，所以无法使用。 调用pip install fabric3 安装Fabric3。 安装好后调用far serve出错，发现没有SocketServer模块，python3中应该是socketserver,改掉即可。 关于voce主题 哇，第一眼就被这个简洁的风格吸引了，加上原博主的那个二位码的logo，感觉不能再好了。之后去github上看了下，并没有很火的样子。嗯，小众，典雅，就决定是你了。然而原来的创作者很久没维护了，落下了很多坑。 Assets插件版本过旧 使用pip install pelican …
